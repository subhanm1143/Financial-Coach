Smart Financial Coach is an AI-powered personal finance dashboard that transforms raw transaction data into clear, actionable insights.
Users can upload bank transaction CSVs, visualize spending, track goals, detect recurring subscriptions, and receive personalized financial coaching generated by AI.

This project was built for a hackathon challenge focused on financial literacy, behavior change, and AI-driven insights.

âš™ï¸ Installation & Setup
1. Clone Repository
git clone https://github.com/yourusername/smart-financial-coach.git
cd smart-financial-coach

2. Install Backend
cd server
npm install

3. Create a .env File

Inside /server:

MONGO_URI=your_mongo_atlas_uri_here
HF_MODEL_ID=meta-llama/Meta-Llama-3-8B-Instruct
HF_TOKEN=your_huggingface_token
PORT=5000

4. Run the Backend
node server.js

5. Install Frontend
cd ../client
npm install
npm start
